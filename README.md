# Tanaka's Gallery [[Live](https://tanaka-gallery.vercel.app/)]

A minimal gallery curation site for fictional Japanese art collector Rin Tanaka.

Made in Japan. Curated in SF.

<img width="1362" height="817" alt="image" src="https://github.com/user-attachments/assets/f66cd982-6355-47ec-9db6-d3cff202439f" />

Next.js (App Router), React, TypeScript, Framer Motion. Checkout is **Stripe test mode only**.

## Run

```bash
npm install
cp .env.example .env.local  # add your Stripe TEST keys (sk_test_/pk_test_)
npm run dev
```
