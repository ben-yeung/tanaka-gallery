# Tanaka Projects — gallery

A minimal gallery curation site for fictional Japanese art collector Rin Tanaka.

Made in Japan. Curated in SF.

Next.js (App Router), React, TypeScript, Framer Motion. Checkout is **Stripe test mode only** — no real payments.

## Run

```bash
npm install
cp .env.example .env.local  # add your Stripe TEST keys (sk_test_/pk_test_)
npm run dev
```
